# BankAppSpring
Java ve Spring Boot kullanarak,hesap açma,para gönderme ,para çekme ve kullanıcı hesap bilgileri görüntülemesi yapılabilen bir backend projesi
# Kullanılan Teknolojiler
* Java
* Spring Boot
* Mysql
* Postman
* Spring Web
* Spring Data JPA
* Lombok
